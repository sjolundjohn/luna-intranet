Brand Guidelines

← UX Review Kit

The Luna iOS app visual language. Calm, precise, trustworthy — the watchful nightlight. D-DIN everywhere; Midnight + Moonlight lead; Plum is the only "critical," never red.

Logo

Primary wordmark with moon phases. Prefer dark surfaces; never stretch, shear, or alter.

Usage rules

  • Primary mark (full wordmark + moon phases) above all others when in doubt.
  • Secondary mark (moon glyph only) only alongside a piece already carrying the primary mark.
  • Placement: upper third for a guardianship feel; prefer dark backgrounds.
  • Clear space = height of the "a" in the wordmark. Min size 40px wide.
  • Never stretch, squeeze, shear, recolor, or alter letterforms / moon icons.

Color

Midnight 40–50% · Paper/Cloud/Periwinkle 35–45% · Moonlight ≤7% · Lunar Lavender ≤5%.

Color guardrails

  • No reds or harsh oranges — Critical = Plum Depth #8E3655, kept rare and significant.
  • No neon / glow gradients. No red/green-only encoding — always pair color with icon or label.
  • Balance Midnight with Paper White / Periwinkle so the UI never reads cold.

Typography

D-DIN, full 11-level iOS HIG Dynamic Type scale. Headline is the 17pt Semibold cut.

Spacing · Radii · Elevation

8pt base (4pt half-step). Screen margins = sm (16). Card padding = md (24). Shadows never layered.

Voice & Imagery

Voice — Quiet · Calm · Protector · Sleep

  • Frame everything through overnight automation: rest → morning outcome.
  • Sensor-agnostic language — never hardcode a CGM brand.
  • Regulatory-safe: no clinical claims, no PHI, no EHR logos in UI.
  • Guiding, never alarming — Plum is serious, not panicked. Approachable, not "high-tech."
  • Show the solution, not the problem — the product working, not glucose anxiety.

Imagery

Soft, diffused dawn/dusk or blue-hour light; bedside & in-hand contexts; clean clinic environments. Avoid bright sterile rooms, staged lab-coat poses, cluttered nightstands, saturated warm tones.
