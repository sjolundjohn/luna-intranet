A shared agent like Basal is a generalist. For some jobs, a specialist is better — an agent that has been given the context, tone, and tools specific to a particular role or workflow.

## When to ask for one

Consider a personal agent when:

- **You run the same kind of task many times a week.** Drafting stand-up notes, triaging a support queue, summarizing weekly reports.
- **The work has strong conventions.** Specific language, formatting, or templates that a generalist keeps drifting away from.
- **You've built up tribal knowledge.** Guidance, decisions, or context that would take paragraphs to re-explain every time.

## What you get

A dedicated agent with its own system prompt, its own conversation memory, and eventually access to tools specific to your work (your calendar, your team's documents, your customer CRM, etc.).

## How to request one

We don't have a formal request flow yet. For now, message **@john** in Slack with:

- What role or workflow the agent is for.
- Three or four example tasks you'd use it for.
- Any tools or data sources it would need access to.

We'll scope it with you, build it, and let you know when it's live.
