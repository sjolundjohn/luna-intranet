The Data Agent is the next shared agent on our roadmap. It sits inside Luna's data visualization tool so that anyone — not just analysts — can ask questions of our data in natural language and get a chart, a number, or a narrative answer back.

## What it will do

- Answer questions like _"How many new users signed up last week?"_ or _"Show me our 30-day retention by cohort"_ without anyone needing to write a SQL query.
- Pull up existing dashboards by description.
- Explain what a chart means in context — the trend, the outliers, the caveats.

## What it won't do on day one

The Data Agent will only see the same data any employee can see today in the data-viz tool. It won't unlock new datasets or bypass permissions. It's a better interface to data you can already access — not new access.

Not yet wired up. We'll announce here when it's live.
